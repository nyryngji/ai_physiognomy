## 관상학 : 자신과 닮은꼴 매칭 서비스

> 개발 기간 
- 2024.05.10 ~ 2024.06.14
<br>

> Stack
- **AI** : Python, Tensorflow, Pytorch
- **FrontEnd** :  HTML, CSS, JavaScript
- **BackEnd** : Django

<br>

> 담당 업무
- Python을 활용하여 강아지 이미지 웹 크롤링
- Tensorflow을 활용한 Inception V3 모델 전이 학습
- HTML, CSS, JavaScript를 활용한 웹페이지 구현
- Django를 활용하여 로컬 웹서버 구현
